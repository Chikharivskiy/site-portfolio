Про мене

Всім привіт, як мене звати ви вже знаєте з першої сторінки на сайті. Також знаєте чим я займаюся. А якщо не знаєте, так вот, звати мене Чихарівський Максим, я верстаю сайти по PSD макатем. Зв'язатися зі мною можна зручним для вас способом по навігації вище. Або нажати на зв'язатися.

Для тих кому цікаво, як я верстаю сайти.

В своїй роботі використовую редактор VS Code. HTML, HTML5 для розмітки сторінки, CSS, CSS3 для оформлення сторінки разом з Sass, JavaScript для інтерактивності сторінки. При завершенні роботи зжимаю всі файли і зображення для швидкодії сайту. Також перевіряю швидкість сайту на Google Page Speed. На виході, сайт одинаково відображеється в ріхних браузерах(кроссбраузерність), а також підлаштовується під різні розміри екрана, тобто відображається як і на комп'ютерах так і на сматрфонах чи планшетах(адаптивність).
